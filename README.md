# to-do-cli
A simple To-Do List CLI application written in Python.

## Installation
**IMPORTANT:** Ensure that you have Python 3.12.3 or higher installed before trying to run this.

To run this program, download the .zip file and unzip it to your desired directory.

## Operation
Once complete, navigate to the application directory in your terminal/command processor and run the following command:
Linux: python3 app.py [action] [data]
Windows: py app.py [action] [data]

Available actions are add, list, and delete.
The data argument for 'add' is the task you want to add.
The 'list' command requires no data argument.
The 'delete' command requires the ID number of the task you want to delete (which will be listed with the 'list' command).
